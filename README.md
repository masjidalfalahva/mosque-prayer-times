# Masjid Al-Falah Prayer Times Plugin

A custom WordPress plugin for displaying daily prayer and iqamah times for Masjid Al-Falah.

## Features

- Daily prayer times
- Iqamah times
- Blue and gold theme
- Ramadan schedule support
- WordPress shortcode
- Admin management (coming soon)

## Installation

1. Download the plugin.
2. Upload it to `/wp-content/plugins/`.
3. Activate the plugin from the WordPress Admin panel.
4. Add the shortcode:

```
[masjid_prayer_times]
```

to any page or post.
